# `VitePress`를 알아 보아요
## `VitePress`?

- `Vue.js` 기반의 SSG(Static-Site-Generation)로써 Vue.js 공식문서에서도 쓰이고 있습니다.
- [공식 문서](https://vitepress.dev/)

## `VitePress` 특징
1. Markdown을 사용
2. 가볍고 빠른 렌더링속도
3. 간단한 설치
4. 페이지 새로고침 없이 바로 수정내용이 반영됩니다.
